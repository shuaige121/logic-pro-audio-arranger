# MelodyLogicBuilder macOS Build

This folder is generated by `scripts/build_macos_app.sh`.

## Contents

- `MelodyLogicBuilder.app`: GUI app for generating Logic import kits.
- `open_terminal_cli.command`: Terminal launcher for CLI mode.

## Usage

1. Open `MelodyLogicBuilder.app`.
2. Choose a melody file (`.wav/.aiff/.mid/.musicxml/.csv`).
3. Click **Generate Logic Kit**.
4. In output bundle, run `open_in_logic.command` on macOS to open Logic Pro with generated arrangement MIDI.

## Notes

- If macOS blocks the app, allow it in **System Settings > Privacy & Security**.
- Logic Pro itself is required to create/save `.logicx` projects.
